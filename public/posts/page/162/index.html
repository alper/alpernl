Apps voor Nederland

So this was announced: A Netherlands wide apps competition called Apps voor Nederland was announced by several ministries today at PICNIC. Minister for the interior Donner delivered a video message as well. The organization I’m involved with, Hack de Overheid will be organizing two events end of this year with the first one being an open hacking event on October 4th and the second being a Code Camp in Amsterdam end of November. The message I am promoting is that this is the time to get on board and get your data out and start building civic apps. This is the last moment in the cycle that any of this is going to be at all special, innovative or unboundedly fun. After we’re done with this event most of the ground will be covered, hearts will be won and anybody talking about open data will be hailed with a ‘Been there, done that!’ ...
