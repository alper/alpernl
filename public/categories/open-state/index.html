My small project during Swhack was to create a django version of a delegated voting system partially inspired by Liquid Feedback and the manyfold problems that system has ((Among others: the atrocious user experience, the fact that it does not support anonymity, the impossibility of i18n.)). In particular that it is written on such an esoteric stack ((The core is a 4500 line pgsql file which is a solid way to deter outside involvement in a project.)) that it is near impossible to get running without root on a Linux machine and let’s not even discuss the maintenance.
