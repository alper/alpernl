[Safari](http://www.apple.com/macosx/features/safari/) trekt de [Acid2 test](http://www.webstandards.org/act/acid2/test.html) nu door een [heldhaftige inspanning](http://weblogs.mozillazine.org/hyatt/archives/2005_04.html#008042) van Dave Hyatt (oud-Mozilla-hacker).

Leukste vind ik nog dat hij de diffs [erbij](http://weblogs.mozillazine.org/hyatt/archives/2005_04.html#008042) heeft geplakt. De C++ code die achter het renderen steekt kan ik een stuk beter lezen dan ik had gedacht. Dit kan zijn omdat ze bij Safari f'ing goede code produceren, of omdat ik beter ben geworden in het lezen van C++.

Ik denk allebei een beetje en vooral dat laatste vind ik leuk.
