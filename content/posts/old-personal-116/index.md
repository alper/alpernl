Python is **da bomb!**

Er komen nog steeds (meer) nieuwe features bij. De nieuwste ervan zijn *decorators*. Redelijk vaag concept met veel flexibel nut.

Het ziet er zo uit:

@foo

def bar()

wat vertaalt in:

def bar()

bar = foo(bar)

Wat heb je hieraan? Heel veel dingen, die niet allemaal direct duidelijk zijn.

Je kunt er bijvoorbeeld een helderder soort aspect georienteerd programmeren mee uitvoeren zoals [hier beschreven](http://www.lonelylion.com/pipermail/chipy/2004-December/000534.html).
